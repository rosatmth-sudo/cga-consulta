# Consulta CGA - SEAPE

Sistema de consulta por linguagem natural para planilha de compras.

## Como publicar

1. Suba esta pasta para o GitHub
2. Conecte o repositório no Vercel
3. Configure a variável de ambiente `CLAUDE_API_KEY` com sua chave
4. Pronto!

## Para atualizar a planilha

Substitua o arquivo `data/planilha.csv` e faça novo commit.
